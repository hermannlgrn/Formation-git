# Commande terminal
Afficher la list des fichiers et repertoires 
```
dir
```

Aller sur un repertoire
```
cd
```

Remonter un repertoire
```
cd img
```

Aller sur c:
```
cd\
```

Vider le terminal
```
cls
```
